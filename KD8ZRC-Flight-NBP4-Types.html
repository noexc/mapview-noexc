This instance is used for saving machine-readable history of packets.

See KD8ZRC.Flight.NBP4.History for more information.

NOTE: The JSON encoding of a TelemetryLine is **NOT* isomorphic to TelemetryLine itself. Notably, we cannot recover a full TelemetryLine because we drop the _rawLine when we serialize above. Here we accommodate for that by using mempty for ByteString.
